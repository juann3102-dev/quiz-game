# 출력 함수 정리
----
## 시작 화면 구성 : Start_screen();
----
1. 시작 화면을 출력하는 함수로 퀴즈를 시작하거나 프로그램을 종료하는 것을 선택할 수 있다.

2. 프로그램 시작 과 종료는 숫자키를 입력하여 선택 가능하다.

3. 만일 숫자가 아닌 다른 키를 입력하였을 때 잘못 입력되었다고 명시되며 다시 입력을 요구한다.

4. 함수의 반환 값은 int 형으로 1(시작) 또는 2(종료)를 반환한다. 

5. system("cis")를 활용하여 터미널을 비워 다음 화면으로 넘어가는 것처럼 연출

----
## 퀴즈 실행 함수 : runQuiz();
----
1. quizNum으로 부터 퀴즈의 갯수를 입력받아 만약 퀴즈의 갯수가 0보다 작으면 반환

2. for문을 활용하여 퀴즈의 개수만큼 getQuiz()를 활용하여 퀴즈 가져오기 

3. checkAnswer()를 활용하여 퀴즈의 정답 비교
